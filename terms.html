4. How the Platform Works

4.1 Posting Tasks
Posters create task listings describing the work needed, the location, budget, and task mode (Quick, Standard, or Recurring). The minimum task budget is $20 CAD.

4.2 Applying for Tasks
Workers browse available tasks and submit applications including their bid price and a message to the Poster. Submitting an application does not guarantee acceptance.

4.3 Accepting Applications
Posters review applications and select one Worker. Once a Worker is accepted, payment is required before any work begins and before the chat between Poster and Worker is unlocked.

4.4 Escrow and Payment
Upon acceptance, the Poster's payment is held in escrow via Stripe. Funds are released to the Worker only after the Poster confirms the task is complete. QuickGigs retains a platform fee of 25% of the task value (20% for subscribers, 15% for top-rated Workers).

4.5 Task Completion
Once the Poster confirms completion, Stripe releases the Worker's payout. Both parties are then invited to leave a review.
